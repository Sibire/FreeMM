═══════════════ Per-Joint Wiring (×5, same pattern) ════════════════

                    TCA9548A                    AS5600 (0x36)
                    CH(n) SDA ──── 2C shielded ───► SDA
                    CH(n) SCL ──── cable ──────────► SCL
                                                    DIR → GND (CW = increasing)
                                                    OUT → float
                                                    GPO → float

                    Separate power bus (unshielded 2-wire):
                    3V3 ─────────────────────────► VCC
                    GND ─────────────────────────► GND


═══════════════════ Cable Lengths (approx) ═════════════════

        Joint    Mux Ch    Signal Cable    Power Cable
        ─────    ──────    ────────────    ───────────
         J1       CH0        ~50mm          ~50mm
         J2       CH1       ~200mm         ~200mm
         J3       CH2       ~400mm         ~400mm
         J4       CH3       ~550mm         ~550mm
         J5       CH4       ~700mm         ~700mm

        Signal cables: 2-conductor SHIELDED (shield grounded at mux end ONLY)
        Power cables:  2-conductor unshielded


═══════════════════════ Buttons ════════════════════════════

        Sample (GPIO 26) ──┤├── GND      On base plate
        Mode   (GPIO 27) ──┤├── GND      On base plate
        Foot   (GPIO 25) ──┤├── GND      Separate pedal, wired to base

        All N.O. momentary, firmware uses INPUT_PULLUP (internal ~45kΩ)
        Press = pin goes LOW, release = pin goes HIGH


══════════════════════ Power Budget ════════════════════════

        ESP32 3.3V regulator (500mA max)
        ├── ESP32 core:         ~40mA
        ├── SSD1306 OLED:        ~8mA
        ├── TCA9548A mux:        ~1mA
        ├── AS5600 × 5:         ~35mA  (7mA each)
        └── Total:              ~63mA  ✓ well within budget


══════════════════════════ Notes ═══════════════════════════

• I2C clock: 400 kHz (set in firmware)
• If long cable runs cause I2C errors, add 4.7kΩ pull-ups
  on SDA and SCL at the mux breakout board
• OLED is on the main I2C bus, NOT behind the mux
  (address 0x3C doesn't conflict with mux 0x70 or AS5600 0x36)
• AS5600 raw angle register: 0x0E (12-bit, big-endian)
• Mux channel select: write (1 << channel) to 0x70
